# Demosaic tool for unity games with edit a shader(.asset).

It's made with [UnityPy](https://github.com/K0lb3/UnityPy).

### Features
  - Find assets files which has mosaic shaders
  - Set colMask to 0

### Watch out
  - Graphics may become corrupted as the colMask value changes.
  - If the graphics are broken or abnormal, restore them using the backed up files.
  - Some .assets files may produce errors when reading.
